cv.boxPoints

boxPoints Finds the four vertices of a rotated rectangle

points = cv.boxPoints(box)

Input

  • box The input rotated rectangle. A structure with the following fields:
    • center The rectangle mass center [x,y].
    • size Width and height of the rectangle [w,h].
    • angle The rotation angle in a clockwise direction. When the angle is 0, 90, 180, 270 etc., the rectangle becomes an up-right rectangle.

Output

  • points The output array of four vertices of rectangles. A 4-by-2 numeric matrix, each row a point: [x1 y1; x2 y2; x3 y3; x4 y4].

The function finds the four vertices of a rotated rectangle. This function is useful to draw the rectangle. You can also use cv.RotatedRect.points method. Please visit the tutorial on bounding rectangle for more information.
