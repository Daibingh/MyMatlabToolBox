cv.GeneralizedHoughGuil/detect

detect Find template on image

[positions,votes] = hough.detect(image)
[positions,votes] = hough.detect(edges, dx, dy)

Input

  • image input image, 8-bit 1-channel image
  • edges image edges
  • dx image x-derivative of the same size as edges and single-precision floating type.
  • dy image y-derivate of the same size and type as dx.

Output

  • positions Cell array of 4-element vectors, each of the form: [posx, posy, scale, angle]
  • votes Cell array of 3-element vectors, of the same length as positions.
