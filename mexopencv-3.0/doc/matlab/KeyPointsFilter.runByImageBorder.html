cv.KeyPointsFilter.runByImageBorder

runByImageBorder Remove keypoints within borderPixels of an image edge.

keypoints = cv.KeyPointsFilter.runByImageBorder(keypoints, imgSize, borderSize)

Input

  • keypoints input collection of keypoints.
  • imgSize image size [width,height].
  • borderSize border size, should be a positive number.

Output

  • keypoints output filtered keypoints.
