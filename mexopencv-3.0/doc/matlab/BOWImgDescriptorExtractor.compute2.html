cv.BOWImgDescriptorExtractor/compute2

compute2 Computes an image descriptor using the set visual vocabulary

bow = extrctor.compute2(img, keypoints)

Input

  • img Image, for which the descriptor is computed.
  • keypoints Keypoints detected in the input image. It is a struct array that is returned by cv.FeatureDetector.detect.

Output

  • bow Computed output image descriptor. A vector of the same length as the vocabulary dimension.
