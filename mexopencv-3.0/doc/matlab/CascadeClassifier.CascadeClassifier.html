cv.CascadeClassifier/CascadeClassifier

CascadeClassifier Creates a new cascade classifier object

classifier = cv.CascadeClassifier()
classifier = cv.CascadeClassifier(filename)

Input

  • filename Name of the XML file from which the trained classifier is loaded. This is handled by the cv.CascadeClassifier.load method.

Supports HAAR and LBP cascades.
