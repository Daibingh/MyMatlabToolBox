cv.tempfile

tempfile Return name of a temporary file

fname = cv.tempfile()
fname = cv.tempfile('OptionName',optionValue, ...)

Output

  • fname Name of temporary file.

Options

  • Suffix optional suffix (file extension). A '.' is prepended if needed. Not set by default.
