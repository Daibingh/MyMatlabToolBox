cv.PSNR

Computes PSNR image/video quality metric

psnr = cv.PSNR(src1, src2)

Input

  • src1 first image (gray or color), 8-bit integer type.
  • src2 second image of same size and type as src1.

Output

  • psnr Computed signal-to-noise ratio

See http://en.wikipedia.org/wiki/Peak_signal-to-noise_ratio for details.
