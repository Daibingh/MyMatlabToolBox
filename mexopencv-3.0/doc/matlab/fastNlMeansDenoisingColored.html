cv.fastNlMeansDenoisingColored

fastNlMeansDenoisingColored Modification of fastNlMeansDenoising function for colored images

dst = cv.fastNlMeansDenoisingColored(src)
dst = cv.fastNlMeansDenoisingColored(src, 'OptionName',optionValue, ...)

Input

  • src Input 8-bit 3-channel or 4-channel image.

Output

  • dst Output image with the same size and type as src.

Options

  • H Parameter regulating filter strength for luminance component. Bigger H value perfectly removes noise but also removes image details, smaller H value preserves details but also preserves some noise. default 3
  • HColor The same as H but for color components. For most images value equals 10 will be enough to remove colored noise and do not distort colors. default 3
  • TemplateWindowSize Size in pixels of the template patch that is used to compute weights. Should be odd. Recommended value 7 pixels. default 7
  • SearchWindowSize Size in pixels of the window that is used to compute weighted average for given pixel. Should be odd. Affect performance linearly: greater SearchWindowsSize - greater denoising time. Recommended value 21 pixels. default 21
  • FlipChannels whether to flip the order of color channels in input src and output dst, between MATLAB's RGB/RGBA order and OpenCV's BGR/BGRA (input: RGB/RGBA->BGR/BGRA, output: BGR/BGRA->RGB/RGBA). default true

The function converts image to CIELAB colorspace and then separately denoise L and AB components with given H parameters using cv.fastNlMeansDenoising function.
