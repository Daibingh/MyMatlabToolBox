cv.colorChange

colorChange Color Change

dst = cv.colorChange(src, mask)
dst = cv.colorChange(src, mask, 'OptionName',optionValue, ...)

Input

  • src Input 8-bit 3-channel image.
  • mask Input 8-bit 1 or 3-channel image.

Output

  • dst Output image with the same size and type as src.

Options

  • R R-channel multiply factor. default 1.0
  • G G-channel multiply factor. default 1.0
  • B B-channel multiply factor. default 1.0
  • FlipChannels whether to flip the order of color channels in inputs src and mask and output dst, between MATLAB's RGB order and OpenCV's BGR (input: RGB->BGR, output: BGR->RGB). default true

Given an original color image, two differently colored versions of this image can be mixed seamlessly.

Multiplication factors are between 0.5 to 2.5
