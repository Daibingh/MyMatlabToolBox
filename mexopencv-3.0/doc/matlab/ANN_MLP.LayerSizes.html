cv.ANN_MLP/LayerSizes

Integer vector specifying the number of neurons in each layer including the input and output layers of the MLP.

The very first element specifies the number of elements in the input layer (feature dimensions). The last element is the number of elements in the output layer (regression dimensions). For example, [N,1] means two-layer network that takes an N-dimensional vector as an input and output scalar. default empty (which means it must be manually set before training).
