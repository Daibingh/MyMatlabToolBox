cv.LogisticRegression/predict

predict Predicts responses for input samples

[results,f] = model.predict(samples)
[...] = model.predict(..., 'OptionName', optionValue, ...)

Input

  • samples The input data for the prediction algorithm. MxN matrix, where each row contains variables (features) of one object being classified. Should have single data type.

Output

  • results Predicted labels as a column matrix of int32 type.
  • f unused and returns 0.

Options

  • Flags The optional predict flags, model-dependent. Not used. default 0
