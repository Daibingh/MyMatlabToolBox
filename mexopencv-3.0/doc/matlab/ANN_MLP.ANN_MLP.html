cv.ANN_MLP/ANN_MLP

Creates an empty ANN-MLP model

model = cv.ANN_MLP()

Use cv.ANNMLP.train to train the model, or cv.ANNMLP.load to load a pre-trained model. Note that the train method has optional flags.
