cv.fillConvexPoly

fillConvexPoly Fills a convex polygon

img = cv.fillConvexPoly(img, pts)
[...] = cv.fillConvexPoly(..., 'OptionName', optionValue, ...)

Input

  • img Image.
  • pts Polygon vertices, stored in numeric array (Nx2/Nx1x2/1xNx2) or cell-array of 2-element vectors {[x,y], [x,y], ...}. Supports integer int32 2D points.

Output

  • img Output image, same size and type as input img.

Options

  • Color 3-element floating point vector specifying polygon color. default zeros
  • LineType Type of the polygon boundaries. One of 8,4,'AA' (Anti-aliased line). default 8.
  • Shift Number of fractional bits in the vertex coordinates. default 0

The function cv.fillConvexPoly draws a filled convex polygon. This function is much faster than the function cv.fillPoly. It can fill not only convex polygons but any monotonic polygon without self-intersections, that is, a polygon whose contour intersects every horizontal line (scan line) twice at the most (though, its top-most and/or the bottom edge could be horizontal).
