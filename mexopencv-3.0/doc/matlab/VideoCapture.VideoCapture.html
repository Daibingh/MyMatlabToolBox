cv.VideoCapture/VideoCapture

VideoCapture Create a new VideoCapture object

cap = cv.VideoCapture(devid)
cap = cv.VideoCapture(filename)

Input

  • devid id of the opened video capturing device (i.e. a camera index). If there is a single camera connected, just pass 0.
  • filename name of the opened video file (eg. video.avi) or image sequence (eg. img_%02d.jpg, which will read samples like img_00.jpg, img_01.jpg, img_02.jpg, ...)

Creates a new video capture instance. With no argument, it connects to the default camera device found in the system. You can specify camera devices by devid, an integer value starting from 0. You can also specify a filename to open a video file.
